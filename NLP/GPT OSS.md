# GPT OSS

Versiones open source del modelo GPT desarrolladas por comunidades y organizaciones académicas.

## Contexto

Los primeros GPT no tenían código público. Modelos como GPT-Neo, GPT-J y GPT-NeoX llenaron el vacío ofreciendo arquitecturas similares de código abierto.

### Modelos

| Modelo | Tamaño | Desarrollador |
|--------|--------|---------------|
| GPT-Neo | 27B | EleutherAI |
| GPT-J | 6B | EleutherAI |
| GPT-NeoX | 20B | EleutherAI |

### Importancia

- Permiten investigación reproducible
- Facilitan el desarrollo de modelos propios
- Contribuyen al ecosistema de [[LLMs en general]] open source
